# 📋 Session Report App — سیستم گزارش‌نویسی جلسات

یک اپلیکیشن وب برای مدیریت و گزارش‌نویسی جلسات آموزشی، ساخته‌شده با React.

---

## 📌 معرفی پروژه

این سیستم برای موسسات آموزشی طراحی شده و فرآیند گزارش‌نویسی جلسات کلاسی را خودکار می‌کند.
سه پنل مجزا برای سرپرست، معلم و دانش‌آموز وجود دارد و هر نقش دسترسی متفاوتی به اطلاعات دارد.

---

## ✨ امکانات

### پنل سرپرست
- مدیریت معلمان (افزودن، مشاهده پروفایل)
- مدیریت دانش‌آموزان (افزودن، مشاهده پروفایل)
- ایجاد ترم برای هر دانش‌آموز با تعیین معلم، روزهای کلاس و ساعت
- مشاهده تمام جلسات و گزارش‌های هر ترم

### پنل معلم
- مشاهده لیست دانش‌آموزان
- مشاهده جلسات هر دانش‌آموز
- نوشتن و ثبت گزارش برای هر جلسه
- ویرایش تاریخ جلسات

### پنل دانش‌آموز
- مشاهده اطلاعات ترم جاری
- مشاهده تاریخچه تمام ترم‌ها
- مشاهده گزارش جلسات گزارش‌شده

## 🛠 تکنولوژی‌های استفاده‌شده

| بخش | تکنولوژی |
|-----|-----------|
| فریم‌ورک | React 18 |
| استایل | SCSS Modules |
| تاریخ شمسی | jalaali-js |
| مسیریابی | React Router v6 |
| احراز هویت | JWT Token |
| ارتباط با بک‌اند | Fetch API |

---

## 📁 ساختار پروژه

```
src/
├── components/
│   ├── common/        # کامپوننت‌های مشترک (Button، Modal، Badge و ...)
│   └── layouts/       # لایه‌بندی صفحات برای هر نقش
├── context/
│   └── AuthContext    # مدیریت احراز هویت
├── pages/
│   ├── supervisor/    # صفحات پنل سرپرست
│   ├── teacher/       # صفحات پنل معلم
│   └── student/       # صفحات پنل دانش‌آموز
├── services/
│   └── api.js         # لایه ارتباط با بک‌اند
└── utils/
    └── dateUtils.js   # توابع تبدیل تاریخ میلادی/شمسی
```

## 🚀 راه‌اندازی محلی

### پیش‌نیازها
- Node.js نسخه 18 یا بالاتر
- npm

### مراحل نصب

```bash
# کلون کردن پروژه
git clone https://github.com/aminifatemeh/session-report-app.git

# رفتن به پوشه پروژه
cd session-report-app

# نصب پکیج‌ها
npm install

# اجرای پروژه
npm run dev
```

اپ روی `http://localhost:5173` اجرا می‌شود.

| نقش | توضیح |

## 📅 ویژگی‌های تاریخ

- تمام تاریخ‌ها در سیستم به صورت **میلادی** ذخیره می‌شوند
- نمایش تاریخ‌ها در رابط کاربری به صورت **شمسی** است
- تاریخ جلسات به صورت خودکار بر اساس روزهای کلاس محاسبه می‌شود
- معلم می‌تواند تاریخ هر جلسه را ویرایش کند
